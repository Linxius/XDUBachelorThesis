# 使用方法
为Linux/Unix 环境的使用方便在fonts文件夹中是Windows中的一些字体，请自行安装

安装texlive或其他环境

修改chapter文件夹下的文件

`xelatex main` 进行编译，为生成正确目录、引用信息需再编译一次

coverTables中是一些封面的表格包括计划书任务书诚信书等等
